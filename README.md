# IRIS Linear Regression

This project demonstrates Linear Regression using the Iris dataset using Python.

## Project Overview
The model was built using scikit-learn and trained to make predictions on the Iris dataset. The results were visualized using matplotlib.

## Technologies Used
- Python
- scikit-learn
- NumPy
- matplotlib

## Files
- Proj_01_IRIS_Linear_Regression.py – Source code
- IRIS REGRESSION PROJECT.PNG – Output visualization

## Output
The project generates a scatter plot comparing actual and predicted values.
